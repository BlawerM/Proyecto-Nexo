# 💎 NEXO PROJECT 
This project is aimed to be used by me to learn more about Springboot and Angular

## 🕵️ Project especifications
- basic API for the managment of jewelry imitation inventory (bisutería)
- basic web application to consume the API and show it
- Learning project for back and front techs

## 🧑‍💻 Technologies
- Spring boot 4.0.3 (Data JPA, Web, Security)
- Java 25 (LTS)
- SQL Server 2025
- Maven (Dependencies)
- Angular (Not implemented yet)

## 🚩 Main goals
 1. Learning how to build both APIRest endpoints and front Consuming webs
 2. Learning Spring boot framework for java
 3. Learning and understand Typescript thru Angular
 4. Create a functional API with the posibility to be deployed

## ✅ Current features
- Basic CRUD endpoints with POSTMAN (WIP)
- Database conection with SQLServer and posibly PostgreSQL

## 🏗️ Future features
- Complete functional endpoint with verification trhu DTOs
- Oauth 2.0 with JWT
- Functional frontend app with Angular
